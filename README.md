# RiffHub
# CONTEXT I AVAST PROJECTE INTERMODULAR - MÚSICA (RIFFHUB)

## Coworking Musical RIFFHUB

### Context i situació inicial
RIFFHUB és un espai creat per donar servei a bandes i artistes que necessiten lloc per assajar, gravar o provar instruments. El centre disposa d’estudis, sales i material variat, però fins ara tota la gestió s’ha fet de manera manual. No hi ha una xarxa informàtica estructurada, no existeix una web pròpia i les reserves es fan per telèfon o de forma presencial. A més, no hi ha cap sistema per controlar el material, els accessos ni les dades de les bandes. La direcció vol modernitzar el projecte i oferir una experiència més còmoda, ràpida i digital als usuaris, així com millorar l’eficiència interna.

## Abast i objectius
L’objectiu principal és digitalitzar el coworking perquè artistes i bandes puguin gestionar tot el procés de manera autònoma i fàcil.

### Objectius específics
- Dissenyar i implementar una web amb horaris, informació de les sales i sistema de reserves d’instruments i estudis.
- Crear una base de dades per gestionar bandes, reserves, material i registre d’ús.
- Afegir serveis al núvol per còpies de seguretat, documents interns i gestió d’arxius d’àudio.
- Aplicar mesures de protecció bàsiques per mantenir el coworking segur i funcional.

## Solució tècnica proposada
El projecte utilitzarà els coneixements del cicle de SMX.

### Muntatge i manteniment d’equips
- Preparar ordinadors per al personal i punts Wi-Fi per cobrir estudis i sales.

### Xarxes locals
- Crear una xarxa separada amb VLANs, servidor DHCP i control d’accés per personal i usuaris.

### Sistemes operatius
- Configurar un servidor Linux amb Apache, MySQL i serveis FTP per gestionar fitxers.

### Aplicacions web
- Desenvolupar la web de RIFFHUB amb HTML, CSS i PHP, connectada a la base de dades de reserves.

### Serveis en xarxa i cloud
- Sincronitzar dades amb una plataforma com Google Workspace i configurar còpies automàtiques al núvol.

### Seguretat informàtica
- Configurar tallafocs, antivirus bàsic i normes de contrasenyes per evitar riscos.

## Planificació i metodologia
La gestió del projecte es farà amb GitHub en diferents fases:

### Anàlisi i planificació
- Recollir necessitats de bandes, personal i direcció.

### Disseny tècnic
- Crear mapes de xarxa, estructures de servidors i decidir eines.

### Desenvolupament
- Construir la web, configurar servidors i preparar la base de dades.

### Proves i validació
- Revisar funcionament, seguretat i ús del sistema de reserves i la xarxa.

## Conclusions i aprenentatges esperats
Aquest projecte permetrà veure com la tecnologia pot transformar un espai creatiu en un servei modern i ordenat. Aplicarem coneixements reals de SMX, aprendrem a organitzar-nos, treballar en equip i completar un projecte. El resultat final farà que RIFFHUB sigui un lloc eficient, segur i fàcil d’utilitzar per totes les bandes que hi passin.
